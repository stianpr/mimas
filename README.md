# Weather Station powered by Raspberry PI

## Connected Sensors

- DHT22 Humidity
- BMP280 Pressure
- DS18B20 Temperature
- QSFS Wind Speed Anemometer
- QSFX Wind Direction
- TX11 Precipitation
